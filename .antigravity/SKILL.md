# Skill: Local Dev Insights (MCP Server)

## 概要
開発者がローカル環境で行う「思考」や「システム状態」をLLMがコンテキストとして取得し、操作可能にするためのMCP（Model Context Protocol）サーバー。

## 獲得する能力
- **コンテキストの永続化**: 開発中の気づきやTODOをSQLiteに保存し、後でLLMが参照できるようにする。
- **実行環境の可視化**: 現在のCPU負荷や特定のポートの開放状況をLLMが把握できるようにする。
- **高度なファイル検索**: 特定の拡張子に絞ったファイルリストの取得など、標準機能以上の探索を提供する。

## 成功の定義
- `stdio` トランスポートを介して Antigravity と正常に通信できる。
- LLMからの要求に応じて、ローカルDBへの書き込みおよび読み込みが整合性を持って行われる。
- 未定義のツール呼び出しや異常な引数に対して、適切なMCPエラーレスポンスを返す。
